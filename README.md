# Predicting Traffic Congestion

## Overview
This Capstone project focuses on analyzing and predicting traffic congestion using the Metro Interstate Traffic Volume dataset. It involves data cleaning, exploratory data analysis (EDA), and building machine learning models to forecast traffic volume accurately.

## Project Structure
- `traffic_volume_analysis.ipynb`: The main Jupyter Notebook containing all the data preprocessing, analysis, and model training steps.
- `Metro_Interstate_Traffic_Volume.csv`: The raw traffic dataset.
- `cleaned_traffic_data.csv`: The preprocessed data ready for model training.
- `PREDICTING TRAFFIC CONGESTION.v2.pptx`: Presentation slides summarizing the project's objectives, methodologies, and final results.

## Highlights
- **Data Cleaning**: Handled missing information, encoded categorical weather features, and engineered time-based features.
- **Exploratory Data Analysis**: Visualized traffic trends across different hours, days, and weather conditions.
- **Modeling**: Developed predictive models to forecast traffic volume, facilitating better traffic management solutions.
